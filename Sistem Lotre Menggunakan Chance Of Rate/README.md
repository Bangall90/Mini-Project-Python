# 🎰 Sistem Lotre Liburan ke Jogja Menggunakan Chance Of Rate

Lotre Liburan ke Jogja adalah program sederhana berbasis Python yang mensimulasikan permainan lotre (gacha) menggunakan konsep random number generation. Pemain dapat membeli atau memasukkan sejumlah token, kemudian menggunakannya untuk melakukan spin 1x atau spin 10x demi mendapatkan hadiah secara acak.
Program ini dibuat sebagai latihan dasar pemrograman Python dengan memanfaatkan perulangan, percabangan, list, input pengguna, serta modul random.

## ✨ Fitur

- Memasukkan jumlah token sebelum bermain.
- Spin sebanyak 1 kali atau 10 kali sekaligus.
- Sistem hadiah acak menggunakan modul random.
- Berbagai hadiah dengan peluang yang berbeda.
- Permainan dapat dimainkan berulang hingga token habis atau pengguna keluar.

## 🏆 Daftar Hadiah

- Zoonk
- Teh SariWangi 1 renceng + Kopi 1 renceng
- Pulsa Rp100.000
- Liburan ke Jogja (Hadiah Utama)

## 🛠️ Teknologi yang Digunakan
- Python 3
- Modul bawaan random

Project ini menerapkan beberapa konsep dasar Python, di antaranya:

## 🎯 Tujuan Project

Project ini dibuat sebagai media pembelajaran untuk memahami dasar-dasar pemrograman Python melalui simulasi permainan lotre sederhana. Selain melatih logika pemrograman, project ini juga memperkenalkan konsep peluang (probability) dan penggunaan bilangan acak dalam sebuah aplikasi sederhana.
