# 📦 Sistem Gudang Sederhana

Program ini merupakan aplikasi **Sistem Gudang Sederhana** berbasis Python yang dijalankan melalui terminal (CLI). Aplikasi ini memungkinkan pengguna untuk menambahkan data barang beserta jumlah stoknya, melihat daftar barang yang telah tersimpan, serta keluar dari program melalui menu interaktif.

Project ini dibuat sebagai latihan dasar Python untuk memahami penggunaan **list**, **perulangan (`while`)**, **percabangan (`if-elif-else`)**, **input pengguna**, dan **method `append()`** dalam menyimpan data secara sederhana.

## ✨ Fitur

* Menambahkan nama barang ke dalam gudang.
* Menambahkan jumlah stok setiap barang.
* Melihat daftar barang beserta stok yang tersimpan.
* Keluar dari program melalui menu.

## 🛠️ Teknologi

* Python 3
* Terminal / Command Prompt

## 📚 Konsep yang Dipelajari

* Variabel
* List (`list`)
* Method `append()`
* Perulangan `while`
* Percabangan `if`, `elif`, dan `else`
* Input dan output
* Menu interaktif berbasis CLI

## 📌 Catatan

Program ini menyimpan data sementara menggunakan list sehingga seluruh data akan hilang ketika program ditutup. Project ini dibuat untuk tujuan pembelajaran dasar Python dan dapat dikembangkan lebih lanjut, misalnya dengan menambahkan fitur:

* Menghapus barang.
* Mengubah jumlah stok.
* Mencari barang berdasarkan nama.
* Menyimpan data ke file (CSV, JSON, atau database).
* Menampilkan data dalam format tabel.
